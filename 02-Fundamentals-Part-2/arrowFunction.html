Arrow Function

       Arrow functions were introduced in ES6 as a shorter form of function expressions, especially useful
 for simple one-line functions

 Key points about arrow functions:
 . For one parameter and one-line body, you can omit parentheses around parameters and curly
 braces
 . With a one-liner without curly braces, the return is implicit
 . For multiple parameters, you need parentheses: 
(param1, param2) => expression
 . For multiple lines of code, you need curly braces and an explicit 
return
 statement
 . Arrow functions don't have their own 
this
 keyword (important for more advanced JavaScript)
